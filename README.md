# Utils

## Mercurial bash prompt

Based on [git-bash-prompt]() by [jerojasro](https://github.com/jerojasro)

### Usage

Add ``source <REPO>/hg-bash-prompt/prompt.sh`` to your ``~/.bashrc`` file and run ``source ~/.bashrc`` to test it out.
